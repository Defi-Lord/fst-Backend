# FST Backend (Express + Prisma + Postgres)

Features:
- Solana wallet login (nonce + signature -> JWT)
- Role-based access (Admin/User)
- Realm-separated contests (free, weekly, monthly, seasonal)
- Prize pool + Top-8 payout rules per contest
- Leaderboards and participants per contest
- Admin dashboard summary endpoints
- Audit log model
